# App Gestione Ferie 🏖️

Applicazione web per la gestione di ferie, ROL e permessi.

## Funzionalità

- Login e Registrazione
- Dashboard con saldi permessi
- Calendario interattivo per inserimento
- Storico movimenti
- Gestione maturazioni mensili
- Configurazioni personalizzabili

## Deploy su Render

1. Crea database PostgreSQL su Render
2. Esegui schema.sql nel database
3. Crea Web Service collegando GitHub
4. Imposta variabili d'ambiente:
   - DATABASE_URL
   - SECRET_KEY
5. Deploy automatico!

## Utilizzo

1. Registrati
2. Configura i valori di maturazione
3. Aggiungi maturazioni mensili
4. Inserisci permessi utilizzati
5. Monitora i saldi
